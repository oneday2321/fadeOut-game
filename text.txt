游戏规则
如何玩
1. 通过执行主程序启动游戏。
2. 按照屏幕上的说明在游戏中导航。
3. 完成所需目标以推进游戏关卡。

具体说明
- 确保定期保存进度。

项目启动说明
前提条件
- 确保您的计算机上安装了 Python 3.x。
- 必须安装所需的库。使用命令：`pip install -r requirements.txt`。

使用说明
1. 使用以下命令将存储库克隆到本地计算机：
   ```
   git clone <repository-url>
   ```
2. 导航到项目目录：
   ```
   cd <project-directory>
   ```
3. 使用以下命令运行游戏：
   ```
   python main.py
   ```
